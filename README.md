# Project GreenEdge (绿刃计划)

> 用代码，为地球降温。

## 🎯 项目使命
汇聚全球开发者的智慧，用一行代码、一个模型，共同推动工业与生活的切实减碳。

## 🌱 核心原则
- **完全开源**：所有代码、模型、文档均开源
- **零成本启动**：基于 GitHub Pages 免费托管
- **财务透明**：所有收支 48 小时内公示
- **贡献公开**：每位贡献者都在贡献者墙留名

## 📁 项目结构
```

├── finance/

├── challenges/

```

## 🚀 如何参与
1. **贡献代码/模型** → 查看 [CONTRIBUTORS.md](CONTRIBUTORS.md) 并提交你的信息
2. **参与挑战** → 查看 [challenges/](challenges/) 中的具体任务
3. **提供其他支持** → 查看 [finance/ledger.md](finance/ledger.md)
4. **改进网站** → 直接修改代码并提交 Pull Request
