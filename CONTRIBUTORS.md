## 如何加入贡献者墙

欢迎你成为绿刃计划的一员！请在此文件的末尾，按照格式添加你的信息并提交一个 **Pull Request**。我们强烈建议你附带一句你的"减碳承诺"。

### 格式示例

| 名字 (Name) | 国家/地区 (Country) | 专业领域/角色 (Role) | 个人主页/联系方式 (Optional) | 我的减碳承诺 (My Pledge) |
| :--- | :--- | :--- | :--- | :--- |
| 张三 | 中国 | 全栈开发 | [github.com/zhangsan](https://github.com/zhangsan) | 我会监督家中所有电器不处于待机状态。 |
| Maria | Kenya | Data Scientist | - | 我会用AI优化社区水泵的能耗。 |

### PR 提交步骤

1. 点击 Gitee 仓库页面右上角的「编辑」按钮
2. 在此文件末尾按格式添加你的一行
3. 提交时选择「创建新的分支并开始 Pull Request」
4. 等待审核通过即可上墙
